# Lottie Animation Prompt for Onboarding Screen 2

## Refined Prompt for Nano Banana Toolkit

**Task:** Create a seamless looping Lottie animation for TravelSnap's Onboarding Screen 2 that demonstrates the full workflow.

**Animation Sequence (2-3 seconds per step):**

1. **Camera Interface Opening**
   - Realistic mobile camera UI opens
   - Viewfinder active showing a landmark/location

2. **Taking a Photo**
   - User takes a picture
   - Camera shutter animation
   - Processing effect overlay

3. **Processing & Location Detection**
   - Camera processes the image
   - Shows "Street View" of the captured location
   - "Identifying location..." loading animation
   - Displays current location detection result

4. **Scrollable List Reveal**
   - A scrollable list slides up below the detected location
   - List includes these specific items with icons:
     - Nearest Stations
     - Nearby Hotels
     - Food Delivery Options
     - Things to Do Nearby
     - Book a Ride from Here
     - eSIM for this Country
     - Restaurants & Cafes Around You

5. **Station Details & Maps**
   - Tap on "Nearest Stations"
   - Open Station Details view
   - Transition to Maps integration showing the route

**Style & Visuals:**
- **AI-Generated Images:** Animated visuals for each workflow step, scrollable icons for the list.
- **Style:** Clean, modern mobile interface with engaging, professional animation. Photorealistic demonstration. Minimalist design.
- **Transitions:** Smooth, fluid transitions between each step.

**Technical:**
- **Format:** Lottie JSON
- **Loop:** Seamless looping
- **Duration:** ~12-15 seconds total
- **Resolution:** High-resolution mobile interface mockup
